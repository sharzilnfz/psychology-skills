## Stages of Change (Prochaska & DiClemente) — apply in Module 10

### The Five Stages

| Stage | User Mindset | Your Approach |
|-------|-------------|---------------|
| **Precontemplation** | "I don't have a problem" / "It's not that bad" | Raise awareness gently. Do NOT push change. Ask about pros/cons of current behavior. Plant seeds. |
| **Contemplation** | "Maybe I should change... but not sure" | Explore ambivalence. Develop discrepancy. Tip the decisional balance. |
| **Preparation** | "I want to change. How do I start?" | Help plan. Build coping toolkit. Set a start date. Identify support. |
| **Action** | "I'm doing it" | Support, troubleshoot, reinforce. Build relapse prevention plan. |
| **Maintenance** | "I've changed. How do I keep it?" | Strengthen coping. Watch for complacency. Celebrate milestones without over-indexing on streaks. |

### Stage-Matched Interventions

**Precontemplation** — DO:
- Ask about what they enjoy about the behavior (paradoxical — shows you're not judging)
- Explore costs they've noticed (gently, not confrontationally)
- Provide information only when asked

**Precontemplation** — DO NOT:
- Use the word "addiction" unless they do
- Set goals for change
- Express disappointment or frustration
- Compare them to others

**Contemplation** — Key technique: Decisional Balance

```
┌─────────────────────┬─────────────────────┐
│  PROS of current    │  CONS of current    │
│  behavior           │  behavior           │
│                     │                     │
│  (what it gives me) │  (what it costs me) │
├─────────────────────┼─────────────────────┤
│  PROS of changing   │  CONS of changing   │
│                     │                     │
│  (what I'd gain)    │  (what I'd lose)    │
└─────────────────────┴─────────────────────┘
```

## Motivational Interviewing (Miller & Rollnick) — apply in Module 30

### Four Processes

1. **Engaging** — build rapport, establish trust
2. **Focusing** — agree on direction
3. **Evoking** — draw out the user's own motivation (change talk)
4. **Planning** — develop a change plan when ready

### OARS Skills

| Skill | Description | Example |
|-------|-------------|---------|
| **O**pen questions | Questions that can't be answered yes/no | "What concerns you most about your drinking?" |
| **A**ffirmations | Genuine recognition of strengths | "It took courage to bring this up" |
| **R**eflections | Mirror back what you hear (simple and complex) | "It sounds like part of you wants to change, but another part isn't sure it's worth the cost" |
| **S**ummaries | Collect and return what's been said | "So far you've mentioned..." |

### Change Talk vs. Sustain Talk

**Change talk** (DARN-CAT) — reinforce and explore:
- **D**esire: "I want to..."
- **A**bility: "I could..."
- **R**easons: "Because..."
- **N**eed: "I need to..."
- **C**ommitment: "I will..."
- **A**ctivation: "I'm ready to..."
- **T**aking steps: "I already..."

**Sustain talk** — roll with, don't argue:
- "I can't imagine life without it"
- "It's not that bad"
- "I've tried before and failed"

Response to sustain talk: reflect it back, then explore what's behind it.
Never debate it.

### The Righting Reflex

The instinct to fix, advise, or convince. **Resist it.** When you argue
for change, the user argues against it. Let them generate their own
reasons — those are the ones that stick.

## CBT for Addiction — apply in Module 20

### Trigger-Thought-Craving-Behavior Chain

```
TRIGGER            →  THOUGHT              →  CRAVING         →  BEHAVIOR
(external/internal)   (permission-giving)     (physical/mental)   (use/act)

Example:
Friday night alone  → "I deserve to relax"  → Urge to drink   → Opens bottle
Stress at work      → "Just one won't hurt" → Restlessness     → Buys cigarettes
Boredom             → "Nothing else to do"  → Anticipation     → Opens gaming app
```

### Common Permission-Giving Thoughts

| Thought | Challenge |
|---------|-----------|
| "Just this once" | When has it ever been just once? |
| "I deserve this" | What do you *really* deserve? (link to values) |
| "I can control it now" | What makes this time different from the last? |
| "Nobody will know" | You'll know. How does that feel the next morning? |
| "I'll start fresh Monday" | Why does Monday have power that right now doesn't? |
| "It's not that bad" | Compared to what? What would 'that bad' look like? |

### Chain Analysis Template (for Module 20)

For each trigger-behavior chain:

1. **Trigger**: What happened? (be specific — time, place, people, feeling)
2. **Thought**: What went through your mind?
3. **Craving**: What did the urge feel like physically?
4. **Behavior**: What did you do?
5. **Consequence**: What happened after? (immediate AND delayed)
6. **Alternative**: At what point could you have done something different?

## Relapse Prevention (Marlatt) — apply in Module 50

### High-Risk Situation Categories

| Category | Examples |
|----------|---------|
| **Negative emotions** | Anger, anxiety, depression, boredom, loneliness |
| **Social pressure** | Being around users, peer pressure, celebrations |
| **Interpersonal conflict** | Arguments, rejection, criticism |
| **Positive emotions** | Celebration, excitement ("I can handle it now") |
| **Testing control** | "Let me see if I can have just one" |
| **Urges/cravings** | Physical or psychological pulls without clear trigger |

### Lapse vs. Relapse

```
LAPSE (slip)                    RELAPSE (return to pattern)
─────────────                   ──────────────────────────
A single instance               Sustained return to old behavior
Data about what needs work       Pattern re-establishment
NOT a moral failure              Often follows unaddressed lapses
Recovery continues               May need plan revision

Key: The Abstinence Violation Effect (AVE)
"I already slipped, so I might as well go all in"
This thought converts lapses into relapses. Name it. Plan for it.
```

### Emergency Protocol Template

```
IF I lapse, I will:
1. STOP — do not continue. One is not the same as ten.
2. NOTICE — "The AVE is talking. This is a lapse, not a relapse."
3. CALL — [specific person] at [specific number]
4. LEAVE — remove myself from the situation physically
5. REVIEW — what triggered this? Update my high-risk plan.
6. RESTART — tomorrow is not "Day 1 again." It's the next step.
```

## SMART Recovery — apply contextually

### Four-Point Program

1. **Building motivation** — why change? (overlaps with MI)
2. **Coping with urges** — urge surfing, HALT, delay strategies
3. **Managing thoughts and feelings** — CBT-based cognitive work
4. **Living a balanced life** — values-aligned lifestyle design

### Hierarchy of Values Exercise

Rank life areas by importance, then by current time/energy allocation.
The gap between the two rankings reveals where addiction has displaced
what matters most.

## Harm Reduction Principles — apply throughout

1. **Any positive change counts** — abstinence is not the only valid goal
2. **Meet people where they are** — not where you think they should be
3. **Reduce harm first** — safer use > dangerous use, even if use continues
4. **No moral judgment** — addiction is a health issue, not a character flaw
5. **User autonomy** — they set the goals, not you
6. **Progress is non-linear** — setbacks are expected and manageable
